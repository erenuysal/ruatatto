# SEO Rehberi — RuaTattoo

Site SEO altyapısı hazır. Aşağıdaki adımları tamamla.

## 1. Yönetim Panelinden SEO

`/admin` → **Site & Tema** → **SEO Ayarları**

- **Sayfa başlığı:** Google'da görünen başlık (max ~60 karakter)
- **Meta açıklama:** Arama sonuçlarındaki kısa metin (max ~160 karakter)
- **Anahtar kelimeler:** dövme, tattoo, Kdz Ereğli, fine line...
- **OG görseli:** WhatsApp/Instagram'da paylaşımda çıkan görsel (1200×630 ideal)

Kaydet → ana sayfayı yenile.

## 2. Google Search Console

1. https://search.google.com/search-console
2. `https://ruatatto.com` mülkiyetin zaten var
3. **Site Haritaları** → Yeni site haritası ekle:
   ```
   https://ruatatto.com/sitemap.xml
   ```
4. Gönder → birkaç gün içinde sayfalar indekslenmeye başlar

## 3. Google İşletme Profili

1. https://business.google.com
2. "Rua Tattoo" veya "Melike Kipritçi" ara / oluştur
3. Adres: Kdz. Ereğli
4. Website: https://ruatatto.com
5. Instagram ile bağla

## 4. Sosyal Medya

Instagram bio'ya site linki ekle: `https://ruatatto.com`

## 5. Teknik (zaten yapıldı)

- `robots.txt` — arama motorlarına izin veriyor
- `sitemap.xml` — otomatik üretiliyor
- Open Graph / Twitter Card meta etiketleri
- JSON-LD (TattooParlor schema) — Google zengin sonuçlar için
- Canonical URL
- Mobil uyumlu tasarım

## 6. İçerik önerileri (organik trafik)

- Her kategoriye (Small/Medium/Big) düzenli yeni dövme fotoğrafı yükle
- Instagram'dan siteye link ver
- Müşteri yorumlarını iste (Google İşletme)
- Blog eklenebilir: "Fine line dövme bakımı", "İlk dövme rehberi" vb.
